# ⚽ Soccer Academy

A responsive website for a soccer academy, built with HTML, CSS, and Bootstrap. It covers everything a prospective student or parent would want to know: training programs, coaching staff, schedules, and how to get in touch. The site is fully deployed and live.

---

## 🔗 Live Site

👉 [main--soccer-academy-bootstrap.netlify.app](https://main--soccer-academy-bootstrap.netlify.app/)

---

## 📄 What's on the Site

- **Home** - Welcome section with a call to action
- **Programs** - Overview of training programs by age group and skill level
- **Coaches** - Profiles of the coaching staff
- **Schedule** - Training session timetable
- **Contact** - Contact form and location details

## 🚀 Running It Locally

No setup required. Just clone the repo and open the HTML file in your browser.

```bash
git clone https://github.com/radhika-khatri/soccer-academy.git
cd soccer-academy
```

Then double-click `index.html` or open it with your browser directly.
